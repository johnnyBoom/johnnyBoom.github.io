新建博客

hexo n "新建博客" //new
hexo clean
hexo g //generated
hexo s //server

将博客托管到GitHub上

安装git托管插件

cnpm install --save hexo-deployer-git

修改_hexo的_config.yml

deploy:
  type: git
  repo: git@github.com:johnnyBoom/johnnyBoom.github.io.git
  branch: master

部署

hexo d

更换主题

下载主题包

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

更改_config.yml

theme: yilia

分类/标签默认页

hexo new page categories
hexo new page tags
---
title: categories
type: categories
layout: "categories"
---

Search功能

$ npm install hexo-generator-searchdb --save

修改hexo的_config.yml

# Hexo-generator-search
search:
  path: search.xml
  field: post

RSS

$ npm install hexo-generator-feed --save
feed:
  type: atom
  path: atom.xml
  limit: 20
  hub:
  content:
  content_limit: 140
  content_limit_delim: ' '
  order_by: -date

配置域名

注册域名

做域名映射
johnnyb.club johnnyBoom.github.io

新建CNAME

根目录新建CNAME johnnyb.club

结合typora插入图片

文件->偏好设置->

格式->图像->设置图片根目录,将hexo/source作为其根目录即可

Code highlight

npm i -S hexo-prism-plugin

修改hexo的_config.yml

highlight:
  enable: false

prism_plugin:
  mode: 'preprocess'    # realtime/preprocess
  theme: 'tomorrow'
  line_number: false    # default false
  custom_css:

word count

npm i --save hexo-wordcount

修改theme的config

postInfo:
  date: true
  update: false
  wordCount: false # set true.
  totalCount: false # set true.
  min2read: false # set true.
  readCount: false # set true.

添加墨迹表情

npm install hexo-filter-github-emojis --save

修改hexo的_config.yml

githubEmojis:
  enable: true
  className: github-emoji
  inject: true
  styles:
  customEmojis:
